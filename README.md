# Android版玩安卓

-   语言：Kotlin
-   架构：MVVM-Clean
-   网络：OkHttp+Retrofitt
-   数据存储：DataStore+MMKV
-   json 解析：Gson+GsonFactory
-   图片加载：Coil
-   列表：SmartRefreshLayout+BaseRecyclerViewAdapterHelper4
-   提示：Toaster
-   轮播图：bannerviewpager
-   WebView：AgentWeb
